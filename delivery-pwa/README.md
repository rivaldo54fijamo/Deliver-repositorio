# 🚀 Delivery PWA System

Sistema completo de delivery de encomendas via motociclistas.

## Tecnologias
- PHP 8+
- MySQL
- HTML5
- CSS3
- JavaScript puro
- Leaflet (OpenStreetMap)
- PWA (Service Worker + Manifest)

## Funcionalidades futuras
- Login multi-perfil
- Cliente, Motorista, Admin e Super Admin
- GPS em tempo real
- Mapa interativo
- Sistema de pedidos
- Notificações em tempo real
- Painéis administrativos
- Relatórios PDF/Excel

🚧 Em desenvolvimento (Sprint 1)
